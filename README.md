# Superstore Sales Dashboard

An interactive sales analytics dashboard built using Power BI.

## Visuals Included
- KPI Cards: Total Sales, Profit and Orders
- Sales by Category (Bar Chart)
- Sales Trend by Month (Line Chart)
- Profit by State (Map Visual)
- Region Slicer for interactivity

## Dataset
Sample Superstore dataset from Kaggle

## Tools Used
Power BI Desktop# superstore-sales-dashboard
